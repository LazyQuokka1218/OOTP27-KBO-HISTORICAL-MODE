# 기여 가이드

KBO Historical Mode 개선에 도움이 되는 제안과 수정은 언제든 환영합니다.

## 기여할 수 있는 것

- 잘못된 데이터 수정
- 누락된 선수, 팀, 시즌 정보 제보
- 로고, 유니폼, 구장, 페이스젠 자산 보완
- 설치 문서, 설명, 번역 개선
- 재현 가능한 버그 제보

## 이슈 제보 방법

이슈를 올릴 때 아래 정보를 함께 적어주면 확인이 훨씬 빨라집니다.

- OOTP 버전
- 적용한 파일 또는 폴더
- 문제가 발생한 시점
- 재현 방법
- 가능하면 스크린샷 또는 로그

## 풀 리퀘스트 가이드

1. 변경 목적을 짧게 설명합니다.
2. 어떤 파일이 바뀌는지 명확히 적습니다.
3. 데이터 수정이라면 근거 자료를 함께 남깁니다.
4. 자산 파일을 바꿨다면 게임 내 확인 결과를 덧붙입니다.

## 권장 원칙

- 기존 구조와 파일명을 최대한 유지합니다.
- 대규모 변경은 작은 단위로 나눠서 제출합니다.
- 출처가 있는 데이터는 가능한 한 출처를 기록합니다.
