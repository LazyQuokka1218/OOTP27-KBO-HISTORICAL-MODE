# Contributing Guide

Contributions to improve KBO Historical Mode are welcome.

## Ways to Contribute

- Fix incorrect data
- Report missing players, teams, or seasons
- Improve logos, uniforms, ballparks, or facegen assets
- Improve installation docs, explanations, or translations
- Report reproducible bugs

## Reporting Issues

Please include the following whenever possible:

- OOTP version
- Files or folders involved
- When the problem happens
- Steps to reproduce
- Screenshots or logs if available

## Pull Request Guidelines

1. Briefly explain the purpose of the change.
2. Clearly list which files were modified.
3. Include source references for data corrections when possible.
4. If visual assets changed, include a quick in-game verification note.

## Recommended Principles

- Keep the existing structure and file naming where possible.
- Split large changes into smaller, reviewable updates.
- Record sources whenever you use externally verified data.
