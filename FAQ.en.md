# FAQ

## Can I use this with an existing save?

You can, but starting from a fresh save is recommended. Existing saves may not pick up all data or asset changes cleanly.

## Which folders do I actually need?

That depends on your setup, but most users will use `database`, `logos`, `uniforms`, `facegen`, `ballparks`, and `stats`.

## Is it safe to overwrite existing files?

Yes, but backing up your original files first is strongly recommended.

## Where should I report bugs or missing data?

Please use the repository Issues page and follow the contribution guide when possible.

## Where are the source or working files?

This repository is intended for release use. Source and working files are maintained in a separate working repository.
